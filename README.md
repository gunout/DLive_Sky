# DLive_Sky
Dashboard en temps réel pour l'audience live de Skyrock avec simulation de données en direct

# INSTALL DEPENDENCIES 

    pip install streamlit pandas numpy matplotlib seaborn plotly 

# RUN PROGRAM

    streamlit run DLive.py
